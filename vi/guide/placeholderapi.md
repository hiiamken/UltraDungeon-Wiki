# PlaceholderAPI

## Tổng quan

UltraDungeon đăng ký placeholder với [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) cho scoreboard, hologram, và plugin khác.

## Placeholder có sẵn

| Placeholder | Mô tả |
|---|---|
| `%ultradungeon_points%` | Tổng điểm |
| `%ultradungeon_season_points%` | Điểm season hiện tại |
| `%ultradungeon_dungeon%` | Tên dungeon đang chơi |
| `%ultradungeon_status%` | Trạng thái game |
| `%ultradungeon_kills%` | Kills trong session |
| `%ultradungeon_deaths%` | Deaths trong session |
| `%ultradungeon_tickets_<dungeon>%` | Ticket còn lại |
| `%ultradungeon_completions_<dungeon>%` | Tổng completions |
| `%ultradungeon_best_score_<dungeon>%` | Điểm cao nhất |
| `%ultradungeon_streak%` | Chuỗi win hiện tại |
| `%ultradungeon_best_streak%` | Chuỗi win cao nhất |
