# Bắt đầu

## UltraDungeon là gì?

UltraDungeon là plugin dungeon Minecraft cho phép bạn tạo **phó bản wave-based** với boss, phần thưởng, scoring, và progression — tất cả cấu hình qua GUI in-game trực quan.

## Tính năng chính

- **Chiến đấu theo Wave** — Không giới hạn số wave quái + boss wave mỗi dungeon
- **GUI Setup Wizard** — Tạo dungeon không cần chỉnh YAML
- **Hệ thống Phần thưởng** — Items, commands, Vault money, MMOItems, MMOCore XP
- **Scoring Engine** — Kill bonus, speed bonus, streaks, leaderboard
- **Progression** — Yêu cầu mở khóa, level gate, MMOCore class/level
- **Hệ thống Party** — Party tích hợp cho nhóm chơi
- **Hệ thống Ticket** — Giới hạn lượt chơi hàng ngày
- **Hệ thống Season** — Leaderboard theo mùa với auto-rewards
- **Tích hợp** — Vault, MMOItems, MMOCore, ItemsAdder, Oraxen, PlaceholderAPI, MythicMobs

## Bắt đầu nhanh

1. Bỏ `UltraDungeon.jar` vào thư mục `plugins/`
2. Khởi động server — config tự động tạo
3. Chạy `/dungeon create <tên>` để mở setup wizard
4. Cấu hình waves, spawn, border, và rewards trong GUI
5. Nhấn **Create** — dungeon sẵn sàng!

::: tip
Dùng `/dungeon validate <tên>` để tự động kiểm tra cấu hình dungeon cho lỗi hoặc thiếu sót.
:::

## Yêu cầu

| Yêu cầu | Phiên bản |
|---|---|
| Minecraft Server | 1.16+ (Spigot/Paper) |
| Java | 17+ |

## Plugin tùy chọn

| Plugin | Công dụng |
|---|---|
| Vault | Economy (phí vào, tiền thưởng) |
| MMOItems | Item reward tùy chỉnh |
| MMOCore | Yêu cầu level/class progression, XP reward |
| PlaceholderAPI | Placeholder cho plugin khác |
| MythicMobs | Spawn quái tùy chỉnh trong waves |
