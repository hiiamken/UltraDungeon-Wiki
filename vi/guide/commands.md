# Lệnh

Lệnh chính: `/dungeon` (alias: `/ud`)

## Lệnh người chơi

| Lệnh | Mô tả |
|---|---|
| `/dungeon join <tên>` | Vào dungeon |
| `/dungeon leave` | Rời dungeon |
| `/dungeon list` | Mở GUI danh sách dungeon |
| `/dungeon top` | Mở bảng xếp hạng |
| `/dungeon party create` | Tạo party |
| `/dungeon party invite <tên>` | Mời người chơi |
| `/dungeon party accept <tên>` | Chấp nhận lời mời |
| `/dungeon party leave` | Rời party |
| `/dungeon party kick <tên>` | Đuổi thành viên (leader) |
| `/dungeon party disband` | Giải tán party (leader) |
| `/dungeon party info` | Xem thông tin party |

## Lệnh Admin

| Lệnh | Mô tả |
|---|---|
| `/dungeon create <tên>` | Mở wizard tạo dungeon |
| `/dungeon edit <tên>` | Chỉnh sửa dungeon |
| `/dungeon continue <tên>` | Tiếp tục chỉnh sửa |
| `/dungeon delete <tên>` | Xoá dungeon |
| `/dungeon reload` | Reload cấu hình |
| `/dungeon set` | Xác nhận vị trí khi setup |
| `/dungeon validate <tên>` | Chạy auto-validation |

## Lệnh Ticket

| Lệnh | Mô tả |
|---|---|
| `/dungeon ticket give <player> <dungeon> <số>` | Cấp ticket bonus |
| `/dungeon ticket take <player> <dungeon> <số>` | Thu ticket bonus |
| `/dungeon ticket set <player> <dungeon> <số>` | Đặt ticket bonus |
| `/dungeon ticket reset <player> <dungeon>` | Reset ticket hàng ngày |

## Lệnh Point

| Lệnh | Mô tả |
|---|---|
| `/dungeon point give <player> <số>` | Cấp điểm |
| `/dungeon point take <player> <số>` | Trừ điểm |
| `/dungeon point set <player> <số>` | Đặt điểm |
| `/dungeon point reset <player>` | Reset điểm |
