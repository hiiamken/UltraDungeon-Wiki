# Điểm & Bảng xếp hạng

## Công thức

```
Score = (Base + Kill Bonus) × Speed × No-Death × Party × Streak
```

| Thành phần | Mô tả |
|---|---|
| **Base Points** | Điểm random từ config |
| **Kill Bonus** | Điểm mỗi kill + hệ số boss |
| **Speed Bonus** | Hệ số theo thời gian còn lại |
| **No-Death Bonus** | Hệ số nếu 0 death |
| **Party Size Bonus** | Ít người = hệ số cao hơn |
| **Streak Bonus** | Win liên tiếp = hệ số tăng dần |
| **Boss Damage** | Điểm theo damage contribution |
