# Commands

Base command: `/dungeon` (alias: `/ud`)

## Player Commands

| Command | Description |
|---|---|
| `/dungeon join <name>` | Join a dungeon |
| `/dungeon leave` | Leave current dungeon |
| `/dungeon list` | Open dungeon selection GUI |
| `/dungeon top` | Open leaderboard GUI |
| `/dungeon party create` | Create a party |
| `/dungeon party invite <player>` | Invite a player to your party |
| `/dungeon party accept <player>` | Accept a party invite |
| `/dungeon party leave` | Leave your party |
| `/dungeon party kick <player>` | Kick a member (leader only) |
| `/dungeon party disband` | Disband party (leader only) |
| `/dungeon party info` | View party info |

## Admin Commands

| Command | Description |
|---|---|
| `/dungeon create <name>` | Open dungeon creation wizard |
| `/dungeon edit <name>` | Edit an existing dungeon |
| `/dungeon continue <name>` | Continue editing a dungeon |
| `/dungeon delete <name>` | Delete a dungeon |
| `/dungeon reload` | Reload configuration |
| `/dungeon set` | Confirm location during setup |
| `/dungeon validate <name>` | Run auto-validation on a dungeon |

## Ticket Commands

| Command | Description |
|---|---|
| `/dungeon ticket give <player> <dungeon> <amount>` | Give bonus tickets |
| `/dungeon ticket take <player> <dungeon> <amount>` | Remove bonus tickets |
| `/dungeon ticket set <player> <dungeon> <amount>` | Set bonus tickets |
| `/dungeon ticket reset <player> <dungeon>` | Reset daily tickets |

## Point Commands

| Command | Description |
|---|---|
| `/dungeon point give <player> <amount>` | Give points |
| `/dungeon point take <player> <amount>` | Take points |
| `/dungeon point set <player> <amount>` | Set points |
| `/dungeon point reset <player>` | Reset points |
